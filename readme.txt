Plugin for displaying latest news from two famous Bangladeshi news paper Daily Prothom Alo & Daily Star.
== Description ==

Main feature and goal of this plugin 'Latest News Marquee' is to display latest news from two famous Bangladeshi newspaper http://prothom-alo.com/ and http://www.thedailystar.net/.
This plugin is really very easy to use in any blog or website for showing latest news from those newspaper. Just using shortcode this feature can be implemented and can be changed look and feel from admin settings.
Wanna see how it works? Click here: http://hizbul.com/latest-news-marquee/


== Installation ==

Installation is very easy like other wordpress plugin
e.g.

1. Upload `Latest News Marquee` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want `Latest News Marquee` in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates

Shortcodes

<strong>Latest News Marquee for Daily Prothom Alo</strong>
<pre>[latest_news id="1"]</pre>

<strong>Latest News Marquee for Daily Star</strong>
<pre>[latest_news id="2"]</pre>

== Frequently Asked Questions ==

= How do I install this plugin? =
You can install as others regular wordpress plugin. No different way. Please see on installation tab.

= What does the mean of id in shortcode =
id mean news paper id, in here Daily Prhothom Alo mean id = 1 and Daily Star mean id = 2

= How to change news and label color =
From plugin settings option you can change both of this.
